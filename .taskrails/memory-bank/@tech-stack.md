# 技術棧說明 (@tech-stack.md)

> **Last Updated**: 2025-12-24

## Frontend

- **Framework**: React 18+
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Diagramming**: Mermaid.js

## Backend

- **Runtime**: Tauri 2.x
- **Language**: Rust (Edition 2021)
- **Database**: SQLite (rusqlite)
- **WebSocket**: Actix-Web

## Build Tools

- **Frontend**: Vite
- **Backend**: Cargo
- **Packaging**: NSIS / MSI
